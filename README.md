# Kui_Nameplates: Kronos Edition
**Optimized for Kronos & Mirasu's MobHealth3**

This is a specialized fork of Kui_Nameplates for the 1.12 client. It has been hard-coded to support real-time health values via the [MobHealth3: Kronos Edition](https://github.com/Mongrul/MobHealth3-Kronos4-Edition) database.

# Key Improvements

Mobhealth3 Integration

Flicker Fix: Resolved an issue where the Blizzard default name plate would flicker in quickly when initialized.

Combat Stability: Improved the "Combat Toggle" reliability, ensuring nameplates appear and disappear correctly.

Error Fixes: Eliminated the `base out of range` and `tonumber` initialization crashes common in other 1.12 ports.

# Recommended Settings
To see health values ensure your settings are configured as follows:
1. Type `/knp` to open the configuration menu.
2. Go to Health display.
3. Set Hostile health format to `<:c;` (this enables the "Current / Max" display).
4. Enable "Show contextual health" to see full HP totals.
